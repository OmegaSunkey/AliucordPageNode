Plugin List:

Juby210:

1.- BetterCodeBlocks

Makes codeblocks better by adding new languages, improving already existing and more.

2.- BetterStatusIndicators

Adds a lot of customization for status indicators (platform indicators, radial status, etc). It is a rewritten version of Butterfly3ffect's BetterStatus with hopefully less bugs and better code.

3.- CustomNicknameFormat

Allows you to customize nickname format.

4.- CustomTimestamps

Custom timestamps format everywhere.

5.- Experiments

Shows hidden Developer Options tab with Experiments.

6.- MessageLogger

Temporary saves deleted and edited messages. WIP. Released version without all planned features because a lot of people asked.

TODO: https://github.com/Juby210/Aliucord-plugins#messagelogger

7.- MoreSlashCommands

Adds more slash commands. (lenny, mock, spoilerfiles, owoify, upper, lower)

8.- NoAutoReplyMention

Disables reply mention by default.

9.- NoCopyEmbeds

Disables copying text from embeds.

10.- OwoifyEverything

Makes your Discord owo.

11.- PronounDB

Shows other's people pronouns in chat, so your chances of mis-gendering them is low. Service by pronoundb.org.

12.- RemoveZoomLimit

Removes maximum (and minimum) zoom limit.

13.- RestoreImageLinks

Restores image links in chat.

14.- ShowReplyMention

Fixes showing @ in replies.

15.- SquareAvatars

Display square avatars instead of circles.

16.- Tags

Allows you to send custom tags. (port of powercord's tags)

17.- UserDetails

Displays when user created account, joined to server and when sent last message in selected server / dm.

18.- ViewRaw

View & Copy raw message and markdown.

Allows you to open user & server avatars/banners by tapping them, similar to attachments or embeds

Adds a dedicated plugin settings category (quite similar to that of Powercord) in the settings page right below Aliucord's settings.

Simple plugin that makes apngs animate just like gifs

35.- PlayableEmbeds

Makes Youtube and Spotify embeds playable directly inside Discord

36.- FixEmotes

37.- TextFilePreview

38.- NoBurnIn

Removes the top bar from the top of the chat to prevent burn in on amoled screens, as its features can all be accessed elsewhere. Maybe later I'll add more stuff

39.- CursorInput

Makes tapping emojis, usernames, and other similar elements add text at the current cursor position, instead of at the end

40.- RandomEmote

Makes long press on the emoji button send a random emote

41.- NoLinkify

Disables highlighting for phone numbers and addresses in messages

42.- AlwaysAnimate

Allows making server icons and user avatars always animate

43.- RoleColorEverywhere

Changes text color in specific areas to match the users highest role. Currently supports mentions, voice users, and typing users, with options to change whats colored

44.- FolderOpacity

Adds an option to the guild folder settings to set the opacity.

RestartButton

Adds a restart button to the toolbar on the settings page which fully restarts the app.

45.- AvatarMention

Makes tapping a message avatar mention that user.

46.- QuickStar

Adds a button to the message context menu that when clicked a star reaction is added to the message, if permission is allowed in the channel.

47.- DisablePreviewsOnData

Disables media previews when on mobile data to limit data usage. Consider disabling Sync across clients in your Text & Images settings so that it doesn't sync to other devices.

48.- RotatedChat

Rotate your chat! Just set a rotation in the plugin settings.

49.- WiderScrollbar

Allows changing the chat scrollbar size, which can be accessed in the plugin settings. Currently effects the height too.

50.- Token

Adds a slash command, /token that tells you your account token. Do not share the token with anybody.

51.- Weather

Adds a weather slash command to show basic information for a location, like temperature and humidity. Uses wttr.in to fetch data.

52.- System Info

Adds a system-info command to show system information.
